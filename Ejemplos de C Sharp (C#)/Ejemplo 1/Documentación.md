### Notas del proyecto

- El namespace "Ejemplo_1" es un contenedor lógico para organizar el código.
- Dentro de este espacio de nombres, se define una clase llamada "Program".
- La clase "Program" contiene el método "Main", que es el punto de entrada de la aplicación.
- Cuando se ejecuta el programa, se muestra el mensaje "¡Hola, Mundo!" en la consola.
- Este es un ejemplo básico de una aplicación en C#.

La clase Console es parte del espacio de nombres System y proporciona métodos para interactuar con la consola.
En este caso, se utiliza el método WriteLine para imprimir el mensaje "Hello, World!" en la consola.

Un método es un bloque de código que realiza una tarea específica.
En este caso, el método Main es el punto de entrada de la aplicación,
lo que significa que es el primer método que se ejecuta cuando se inicia el programa.

- WriteLine es un método de la clase Console que se utiliza para escribir una línea de texto en la consola.

- El método ReadKey se utiliza para esperar a que el usuario presione una tecla antes de cerrar la consola.

En resumen, este código es un programa simple en C# que muestra unos mensajes en la consola cuando se ejecuta.
